# MediMind AI

## Project Description

MediMind AI is an intelligent medication reminder system that helps users take medicines on time.

The system allows users to add medicines, set reminder times, receive voice reminders, and ask AI questions about medicines.

It also stores medicine history and provides intelligent suggestions using Artificial Intelligence.

## Problem Statement

Many people forget to take medicines at the correct time.

Missing medicines may lead to health problems.

Current reminder applications only notify users without explaining medicines or giving intelligent assistance.

MediMind AI solves this problem using AI-powered reminders and voice assistance.

## Objective

Develop an AI-powered medication reminder that provides:

- Medicine reminders
- Voice notifications
- AI chatbot
- Medicine history
- Prescription management
- User-friendly interface
